# README 截图资源

将界面截图放在本目录，文件名需与下方一致（PNG 或 WebP 均可，扩展名保持一致即可）。

| 文件名 | 说明 | 建议内容 |
|--------|------|----------|
| `servers-list.png` | 服务列表 | `#/` 服务器列表页 |
| `server-detail.png` | 服务管理 | `#/servers/:id` 服务器详情页 |
| `config-editor.png` | 配置编辑 | `#/servers/:id/edit` 配置编辑器（画布或源码均可） |

## 规范

- 宽度建议 1200px 左右，单张尽量 < 500KB
- 更新界面后请同步替换对应截图
